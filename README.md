# 这是一个基于jquery dom渲染的视觉小说游戏框架

# 亮点：
1. 轻量级
2. core.js是它的核心 只有7kb大小
3. 每个功能分模块使用 你可以在module文件夹下面看见不同的模块
4. 灵活性大，每个对象基本都是jquery对象，你可以根据自己的需求自由更改代码，添加代码
5. 运行在浏览器上，跨平台，当然你也可以使用其他工具打包
6. var Engine=new EasyAvg() 使用此方法获得引擎类
